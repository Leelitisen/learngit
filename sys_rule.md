胶粘剂与光学镜面粘接知识库置信度评价系统
1.  概述
1.1 系统定位
本系统是多源摄入知识库的核心评价引擎，用于对单条数据记录进行综合可信度评分，支撑工程决策：胶粘剂选型、光学镜面粘接工艺设计、失效根因分析。
1.2 设计目标
•  可计算：每条数据输出一个 0-1 之间的综合置信度分数，支持排序和筛选
•  可追溯：置信度由四个独立维度乘积构成，每个维度可独立审计
•  可操作：置信度直接对应工程决策门槛（能否纳入工艺库、能否用于选型、仅作参考）
•  可扩展：新增维度时不破坏现有评分体系
1.3 适用范围
•  胶粘剂选型与供应商评估
•  光学镜面粘接工艺设计与验证
•  结构粘接失效分析
•  多源数据（TDS、文献、专利、内部实验、标准规范）的融合与排序

2.  系统架构
2.1 整体架构
┌─────────────────────────────────────────────────────────────────────┐
│                    置信度评价系统 (Confidence Evaluation System)      │
├─────────────────────────────────────────────────────────────────────┤
│  输入层                                                              │
│  ├─ 单条数据记录（胶粘剂型号、基材、工艺参数、性能值、来源文档）      │
│  └─ 用户实际工况（温度、湿度、基材、表面处理、载荷、光学要求）        │
├─────────────────────────────────────────────────────────────────────┤
│  评价层 — 四维独立乘数                                                │
│  ├─ 信源类型 (Source Type)        → 基础可信度权重 W                │
│  ├─ 证据等级 (Evidence Level)      → 验证深度置信分 S               │
│  ├─ 可重复状态 (Reproducibility)   → 操作可行性系数 R               │
│  └─ 工况匹配 (Condition Match)     → 条件匹配系数 C                 │
├─────────────────────────────────────────────────────────────────────┤
│  输出层                                                              │
│  ├─ 综合置信度 Score = W × S × R × C                                │
│  ├─ 各维度分项得分（用于诊断）                                        │
│  ├─ 冲突标记（如有）                                                  │
│  └─ 工程决策建议（纳入工艺库 / 选型参考 / 仅作情报）                  │
└─────────────────────────────────────────────────────────────────────┘
2.2 四维评价模型
四个维度正交独立，各自回答不同的问题：
维度	符号	回答的问题	取值范围
信源类型	W	数据从哪来？这个来源有什么先验偏见？	0.30 — 0.95
证据等级	S	数据被验证到什么程度？	0.20 — 1.00
可重复状态	R	数据包含的参数足够让我复现吗？	0.10 — 1.00
工况匹配	C	数据测试条件与我的实际应用匹配吗？	0.20 — 1.00
正交性说明：四个维度是独立的风险因子，不是互斥分类。例如：
•  TDS（W=0.85）可以是 E5（S=0.40）或 E3（S=0.70）
•  文献（W=0.60）可以是 R5（R=1.00）或 R3（R=0.60）
•  内部实验（W=0.95）可以是 MATCH-EXACT（C=1.00）或 MATCH-EXTRAPOLATE（C=0.50）
2.3 数据流与处理流程
Step 1: 数据捕获
├─ 来源：TDS PDF、文献 PDF、内部 ELN、专利、标准文档
└─ 动作：保留原始文件，计算哈希，记录捕获时间
Step 2: 结构化解析
├─ 来源：大模型自动提取 + 人工校验
└─ 输出：JSON 结构化记录，含材料、工艺、性能、来源元数据
Step 3: 四维标注
├─ 信源类型：根据来源文档类型自动判定
├─ 证据等级：根据数据生成方式判定
├─ 可重复状态：对照 R5-FULL 清单逐项审计
└─ 工况匹配：对比数据测试条件与用户实际工况
Step 4: 置信度计算
├─ 公式：Score = W × S × R × C
├─ 冲突检测：触发 CONFLICT-VAL / CONFLICT-CTE 等标记
└─ 输出：综合置信度 + 分项得分 + 冲突标记
Step 5: 质量门控
├─ Score ≥ 0.70 → 纳入工艺库
├─ Score ≥ 0.50 → 纳入选型库
├─ Score ≥ 0.30 → 纳入文献综述
└─ Score < 0.30 或冲突标记触发 → 人工复核
Step 6: 知识融合
├─ 通过门控的数据进入主知识库
├─ 建立实体链接（胶粘剂-基材-工艺-性能）
└─ 记录完整溯源链，设置复核日期

3.  系统详细介绍
3.1 信源类型模块
作用：评估数据的"出身"——来源本身有什么系统性偏见。
核心逻辑：不同来源有制度性偏见：TDS 有最优条件偏差，专利有故意模糊化，学术文献有发表偏见，内部实验最接近真实但可能缺乏系统整理。
七类信源及权重：
信源	权重	工程定位
EXP 实验/工艺记录	0.95	工程核心，直接指导工艺
STD 标准/规范	0.88	合规检查依据，最低门槛
TDS 技术数据表	0.85	选型首选，供应商官方数据
DB 数据库/知识库	0.70	结构化参考，需追溯原始来源
LIT 学术文献	0.60	学术参考，需内部验证后使用
PAT 专利文献	0.50	技术情报，不直接用于选型
IND 产业/商业资料	0.30	市场情报，不可用于工艺
详细说明见：《信源类型标准》
----
3.2 证据等级模块
作用：评估数据的"验证深度"——从原始产生到传闻证据的六级梯度。
核心逻辑：数据被验证得越充分，可信度越高。独立复现（E2）过滤了随机误差，原始记录（E1）是证据链起点。
六级体系：
等级	置信分	核心定义
E1 PRIMARY	1.00	原始实验数据，含原始仪器输出
E2 REPLICATED	0.90	独立复现验证
E3 PEER-REVIEWED	0.70	同行评审但未独立复现
E4 PARTIAL	0.50	部分验证/条件受限
E5 SECONDARY	0.40	二次推导/间接证据
E6 ANECDOTAL	0.20	个案/传闻证据
详细说明见：《证据等级标准》
3.3 可重复状态模块
作用：评估数据的"可操作性"——参数是否完整到可以直接用于工艺。
核心逻辑：即使数据来自高可信度来源（EXP 0.95）、经过了独立验证（E2 0.90），如果关键工艺参数缺失，仍然不能用于指导生产。
六级体系：
状态	系数	核心定义
R5 FULL	1.00	完全可重复，参数完整到可直接转化为 SOP
R4 CONDITIONAL	0.85	条件可重复，1-2 个非关键参数缺失
R3 METHOD	0.60	方法可重复，只能复现方法框架
R2 PARTIAL	0.35	部分不可重复，关键信息缺失
R1 NON-REP	0.10	不可重复，信息不足以支持任何复现
R0 PENDING	0.50	待验证，新录入数据尚未验证
详细说明见：《可重复状态标准》
3.4 工况匹配模块
作用：评估数据的"适用性"——测试条件与实际应用场景是否匹配。
核心逻辑：采用木桶效应（取最低分）。如果温度匹配是 MATCH-MISMATCH（0.20），即使其他维度都是 MATCH-EXACT（1.00），整体工况仍然不匹配。
五级体系：
匹配级别	系数	核心定义
MATCH-EXACT	1.00	测试条件与实际应用条件完全一致
MATCH-CLOSE	0.90	关键参数差异 < 10%
MATCH-RELATED	0.75	同系材料或相近条件，需经验推断
MATCH-EXTRAPOLATE	0.50	需要外推，超出测试范围
MATCH-MISMATCH	0.20	测试条件与实际应用严重不匹配
详细说明见：《工况匹配标准》
3.5 各模块的交互关系
四个模块正交独立，通过乘法关系组合：
Score = W_source × S_evidence × R_repro × C_condition
乘法而非加法的理由：
•  各维度是独立的风险因子，不是互斥分类
•  如果任一维度为 0（如工况完全不匹配），整体置信度应为 0，而不是被其他维度稀释
•  保留绝对尺度意义：>0.90 高可信度，0.60-0.90 中等，<0.60 低可信度
4.  评价规则
4.1 综合置信度计算公式
综合置信度 = 信源权重 × 证据等级置信分 × 可重复性系数 × 工况匹配系数
即：Score = W × S × R × C

4.3 质量门控（阈值体系）
应用场景	最低置信度阈值	说明
纳入工艺库（直接指导生产）	≥ 0.70	必须 R4 及以上 + 工况 MATCH-RELATED 及以上
纳入选型库（供应商筛选）	≥ 0.50	允许 R3，但需标注工况匹配度
纳入文献综述（技术调研）	≥ 0.30	允许低置信度，但需明确标注
自动触发人工复核	< 0.30 或冲突标记触发	防止垃圾数据污染
4.4 冲突标记与自动拦截
标记	触发条件	处理策略
CONFLICT-VAL	不同来源强度差异 > 30%	按置信度排序，优先采信高置信度；检查是否为表面处理差异导致
CONFLICT-METH	不同测试方法或不同表面处理导致结果差异	按方法/表面处理分组，不直接合并
CONFLICT-CTE	声称 CTE 匹配但热循环后面形漂移严重	以热循环后面形数据为准，推翻 CTE 匹配声明
CONFLICT-OPT-MECH	光学性能与力学性能冲突（如强度高但面形变化大）	优先采信光学性能数据，力学强度仅作辅助
CONFLICT-SCOPE	声称"通用型"但多基材验证缺失	缩小适用范围，标注"仅验证基材 X"
TDS-UPDATE	TDS 版本已更新，旧版数据可能失效	标记旧版数据，优先使用新版
STD-REVISION	标准版本已更新	标记旧标准数据，优先使用新标准
高声称低证据自动拦截：
•  声称强度远超同类型胶粘剂典型值（如环氧在 PE 上 > 10 MPa 未经表面处理）但 evidence_level < E3，自动触发人工复核
•  声称面形 λ/20 但 evidence_level < E3 且无干涉仪数据，自动触发人工复核
4.5 降级规则
触发条件	降级动作
未报告失效模式	证据等级最高只能到 E4
未报告光学性能（面形/应力双折射）	证据等级最高只能到 E4
未报告表面处理细节	可重复状态最高只能到 R4
未报告胶层厚度实测值	可重复状态最高只能到 R4
未报告固化环境湿度	可重复状态最高只能到 R4
仅有结论性描述无参数	可重复状态最高只能到 R1
工况严重不匹配	直接降级到 MATCH-MISMATCH（0.20）
4.6 升级路径
升级方向	条件
E5 → E4	人工校验关键参数
E4 → E3	补充缺失的关键参数
E3 → E2	独立来源交叉验证，或同一团队 ≥ 3 批次统计重复
E2 → E1	获取原始实验记录
R0 → R4	完成内部实验验证，关键参数完整
R2 → R4	补充实验验证，获取完整工艺参数
R3 → R5	补充全部参数，包括实测值和失效分析
R4 → R5	补充剩余缺失参数，消除推断项
5.  系统边界
5.1 适用边界
本系统适用于以下场景：
•  胶粘剂多源数据（TDS、文献、专利、内部实验、标准）的综合可信度排序
•  光学镜面粘接工艺参数的知识库构建与工艺推荐
•  结构粘接失效分析中的数据可信度评估
•  供应商选型时的数据筛选与优先级排序
5.2 不适用场景
以下场景不适用本系统：
•  纯学术研究中的理论推导可信度评估（本系统面向工程应用）
•  单一来源数据的绝对正确性判定（本系统评估的是相对可信度，不是真理）
•  涉及商业机密保护的数据（如供应商核心配方，本系统不处理解密问题）
•  实时在线监测数据的动态可信度评估（本系统面向静态知识库数据）
5.3 局限性说明
局限性	说明	应对策略
维度独立性假设	四个维度被假设为独立，实际可能存在微弱相关性（如高可信度来源通常参数也更完整）	接受近似，定期校准
系数取值经验性	权重和系数基于工程经验，非严格统计推导	基于内部复现实验进行贝叶斯更新
木桶效应保守性	取最低分可能导致整体评分被单一短板过度拉低	记录分项得分，供人工复核时参考
大模型解析误差	自动提取工具精度约 85%，存在 15% 错误率	自动提取数据初始等级最高 E5，必须人工校验
工况维度不完备	实际工况可能包含本系统未覆盖的参数（如振动、辐射、化学腐蚀）	新增工况维度时按相同原则扩展
5.4 与外部系统的关系
外部系统	关系	交互方式
大模型解析系统	上游输入	接收结构化提取后的 JSON 数据，进行四维标注
知识库主系统	下游输出	通过门控的数据进入主知识库，建立实体链接
实验管理系统（ELN）	平行系统	接收 ELN 原始数据作为 EXP 来源，反馈验证结果升级证据等级
供应商管理系统	平行系统	接收 TDS 数据，反馈版本更新信息触发 TDS-UPDATE 标记
标准管理系统	平行系统	接收 STD 数据，反馈标准修订信息触发 STD-REVISION 标记

6.  工程应用示例
6.1 胶粘剂选型场景
用户提问："我需要粘接强度在 20-30 MPa 的胶粘剂，用于 Al6061 阳极氧化件，25°C/60%RH，结构剪切。"
系统处理：
数据	来源	W	S	R	C	Score	决策
内部 ELN：EPO-TEK 301-2，22.5 MPa	EXP	0.95	1.00	1.00	1.00	0.950	纳入工艺库
文献：某环氧胶，28 MPa，Al6061，"标准清洁"	LIT	0.60	0.70	0.60	0.75	0.189	选型参考
TDS：3M DP460，29 MPa，钢片，lightly abrade	TDS	0.85	0.50	0.35	0.50	0.074	初筛参考
系统回答：
推荐 EPO-TEK 301-2（置信度 0.950），内部验证数据，工况完全匹配。3M DP460 强度虽高但工况错配严重（基材/表面处理差异），仅作初筛参考，需实验验证。
----
6.2 光学镜面粘接场景
用户提问："熔石英镜面（D100mm）与殷钢托片粘接，要求热循环 -40~+60°C 后 PV < λ/10 @ 632.8nm。"
系统处理：
数据	来源	W	S	R	C	Score	决策
内部 ELN：EPO-TEK 301-2，PV=λ/15，热循环后 λ/14	EXP	0.95	1.00	1.00	0.90	0.855	纳入工艺库
文献：UV 胶室温 λ/20，无热循环数据	LIT	0.60	0.70	0.60	0.50	0.126	方法参考
TDS：Norland 61，折射率 1.56，无面形数据	TDS	0.85	0.50	0.35	0.50	0.074	初筛参考
系统回答：
推荐 EPO-TEK 301-2（置信度 0.855），内部验证数据含热循环后光学性能。Norland 61 无面形/热循环数据，仅作光学参数初筛。UV 胶文献仅室温数据，热循环应用需重新验证。
6.3 失效分析场景
用户提问："产线 Al-PC 粘接件在 85°C 老化 500h 后脱粘，请分析可能原因。"
系统处理：
•  检索知识库中所有 Al-PC 粘接 + 高温老化相关数据
•  按置信度排序，优先展示高可信度数据
•  冲突标记检测：某文献声称该胶粘剂耐 120°C，但内部失效分析显示 60°C 即降解 → 触发 CONFLICT-TEMP
系统回答：
高置信度数据（0.855）：内部失效分析显示该胶粘剂 Tg ≈ 65°C，85°C 已远超 Tg，胶层软化导致界面失效。
冲突标记：某文献声称耐 120°C（置信度 0.189），但测试条件为短时暴露，与长期老化工况不匹配。建议以内部失效分析为准，更换 Tg > 100°C 的胶粘剂。
